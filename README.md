# Quarterly Earnings — Reveal.js Presentation

Prepared by: 22f1001941@ds.study.iitm.ac.in

This repo contains a Reveal.js presentation for the Quarterly Earnings Report.

## Publish to GitHub Pages (simple method)
1. Put the presentation files in the `docs/` folder (we used `docs/index.html`).
2. Commit and push to the `main` branch.
3. Go to the repo **Settings → Pages**.
4. Under "Source", select **main branch / docs folder**. Save.
5. After a minute, the site will be available at:
   `https://<GITHUB_USERNAME>.github.io/<REPO_NAME>/`

## Example
If your GitHub username is `alice` and the repo is `quarterly-earnings-presentation`, the URL will be:
`https://alice.github.io/quarterly-earnings-presentation/`
