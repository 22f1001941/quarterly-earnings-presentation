Financial Formula

We estimate **Customer Lifetime Value (LTV)** using:

$$ \mathrm{LTV} = \frac{\text{Average Order Value} \times \text{Purchase Frequency}}{\text{Churn Rate}} $$

Example: AOV = \$50, freq = 3/year, churn = 0.2 → LTV = \( \dfrac{50 \times 3}{0.2} = 750 \)
